Fontes externas

Fontes externas CSS são fontes que podem ser adicionadas a um site usando a regra @font-face do CSS. Para adicionar uma fonte externa, a regra @font-face deve ser incluída no arquivo CSS antes dos outros estilos. A diretriz @font-face cria uma nova família tipográfica que pode ser usada pelo atributo font-family no CSS.
